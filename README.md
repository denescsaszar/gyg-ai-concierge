# AI Travel Concierge for GetYourGuide

An AI-powered travel experience discovery feature that helps travelers find
personalized activities through natural language conversation.

## Tech Stack

- **Backend:** Kotlin + Spring Boot 3
- **Frontend:** Vue 3 + TypeScript + Vite
- **AI:** Claude API (Anthropic) for intelligent recommendations
- **Infrastructure:** Docker Compose

## Architecture

├── backend/ # Kotlin/Spring Boot REST API + AI service
├── frontend/ # Vue 3 + TypeScript SPA
└── docker-compose.yml
